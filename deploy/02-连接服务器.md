# 🔌 连接阿里云服务器

## 💻 Windows 用户连接方式

### 方法一：使用 Windows Terminal + SSH（推荐）
```bash
# 在 PowerShell 或 CMD 中执行
ssh root@你的服务器IP地址

# 例如：
ssh root@47.xxx.xxx.xxx
```

### 方法二：使用 PuTTY
1. 下载 PuTTY: https://www.putty.org/
2. 打开 PuTTY
3. 输入服务器IP地址
4. 端口：22
5. 连接类型：SSH
6. 点击"Open"
7. 输入用户名：root
8. 输入密码

### 方法三：阿里云控制台
1. 登录阿里云控制台
2. 找到您的 ECS 实例
3. 点击"远程连接"
4. 选择"VNC连接"

## 🐧 macOS/Linux 用户
```bash
# 直接在终端执行
ssh root@你的服务器IP地址
```
